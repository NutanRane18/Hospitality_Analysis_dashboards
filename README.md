# 🏨 Hospitality_Analysis_dashboards

## 📊 Project Overview
This project focuses on analyzing hotel performance using data analytics and visualization tools. The goal is to derive actionable insights on revenue, bookings, occupancy, and customer behavior to support business decision-making.

---

## 🚀 Key Highlights
- 📌 Total Bookings: **135K**
- 💰 Total Revenue: **₹1.7 Billion**
- 📉 Cancellation Rate: **25%**
- 🏨 Occupancy Rate: **57.87%**
- 💵 ADR: **₹12.7K**
- 📊 RevPAR: **₹7.35K**

---

## 🛠️ Tools & Technologies
- **SQL (MySQL)** – Data cleaning, transformation & KPI creation  
- **Power BI** – Interactive dashboards & business insights  
- **Tableau** – Advanced visualization & executive reporting  
- **Excel** – Data analysis & dashboarding  

---

## 📈 Key Analysis Performed
- Revenue analysis by **city, hotel, and booking platform**
- Occupancy and capacity utilization analysis
- Cancellation trends and revenue loss analysis
- Customer segmentation (Business vs Luxury)
- Weekday vs Weekend performance analysis
- Booking trends and guest behavior insights

---

## 📊 Dashboards Included

### 🔹 Power BI Dashboard
- Revenue & Performance Overview  
- Booking & Guest Behavior Analysis  

### 🔹 Tableau Dashboard
- Executive Overview  
- Hotel Performance Analysis  

### 🔹 Excel Dashboard
- Hospitality Executive Summary  
- Revenue & Booking Trends  

---

## 💡 Business Insights
- Identified high cancellation rate impacting revenue significantly  
- Mumbai generated the highest revenue among all cities  
- Weekend bookings showed higher occupancy compared to weekdays  
- Certain booking platforms contributed more to revenue generation
